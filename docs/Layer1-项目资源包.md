## 一、资源存储说明

### 1.1 两种存储位置

| 存储位置 | 内容 | 容量限制 | 费用 |
|----------|------|----------|------|

| 类别 | 预算 | 说明 |

## 三、资源规格要求

### 3.1 图片资源

| 资源类型 | 格式 | 建议尺寸 | 文件大小 | 说明 |
|----------|------|----------|----------|------|
| 精灵图 | PNG | 32x32 或 64x64 像素 | < 100KB | 透明背景 |
| 角色头像 | JPG/PNG | 200x200 ~ 400x400 像素 | < 500KB | 正方形，清晰面部 |
| 场景地图 | PNG | 根据设计 | < 5MB | 可以较大 |
| UI 图标 | SVG | 矢量 | < 10KB | 可缩放 |

### 3.2 音频资源

| 资源类型 | 格式 | 建议时长 | 文件大小 | 说明 |
|----------|------|----------|----------|------|
| 背景音乐 | MP3 | 2-5 分钟 | < 10MB | 可循环播放 |
| 音效 | MP3/WAV | < 5 秒 | < 500KB | 可选 |

### 3.3 文本资源

| 资源类型 | 格式 | 说明 |
|----------|------|------|
| 角色 prompt | TXT/MD | 角色人设描述，用于 AI 对话 |
| 配置文件 | TS/JS | 代码中的角色/地图配置 |

## 四、命名规范

### 4.1 文件命名

- 使用**小写英文**
- 单词间用**短横线** `-` 连接
- 避免空格和中文

```
✅ 正确：tong-xiangyu.jpg, background.mp3
❌ 错误：佟湘玉.jpg, Background Music.mp3
```

## 五、注意事项

### 5.1 版权相关

| 注意项 | 说明 |
|--------|------|
| 电视剧截图 | 仅限粉丝二创使用，不可商用 |
| 背景音乐 | 使用电视剧原声需注意版权 |
| 二创标注 | 建议在项目中注明"粉丝二创作品" |

### 5.2 性能相关

| 注意项 | 说明 |
|--------|------|
| 图片压缩 | 上传前压缩，推荐使用 TinyPNG |
| 音频压缩 | MP3 使用 128kbps 即可 |
| 总大小 | 控制在 200MB 以内 |

### 5.3 兼容性相关

| 注意项 | 说明 |
|--------|------|
| 图片格式 | 优先 JPG（照片）、PNG（透明）、SVG（图标） |
| 音频格式 | 优先 MP3（兼容性最好） |
| 避免格式 | WEBP（旧浏览器不支持）、FLAC（太大） |
